Syntax

CSS ruleset has a selector and a declaration block.

selector {
      property: value;
    }

Selectors

  • * - Universal
  • #id - ID Selector
  • .class - Class Selector
  • element - Tag Selector

Common Properties

  • color - Text color
  • background-color - Background color
  • margin - Space outside the element
  • padding - Space inside the element
  • font-size - Size of text

Examples

Here is how you style a paragraph:

p {
      color: blue;
      font-size: 18px;
    }

Flexbox

One‑dimensional layout for rows or columns. Use display: flex; on a container :contentReference[oaicite:10]{index=10}.


.container { 
  display: flex; 
  justify-content: space-between; 
  align-items: center; 
}

Grid

Two‑dimensional layout with rows and columns. Use display: grid; and grid-template shorthand :contentReference[oaicite:11]{index=11}.


.wrapper {
  display: grid;
  grid-template-columns: 1fr 2fr;
  grid-gap: 1rem;
}

Responsive Design

Media queries adapt styles to different screen sizes :contentReference[oaicite:12]{index=12}.


@media (max-width: 600px) {
  .container { flex-direction: column; }
}

CSS Transitions

Smoothly animate property changes with transition :contentReference[oaicite:13]{index=13}.


button {
  background: #007bff;
  transition: background 0.3s ease;
}
button:hover {
  background: #0056b3;
}

CSS Animations

Define keyframes and control timing with the animation shorthand :contentReference[oaicite:14]{index=14}.


@keyframes fadeIn {
  from { opacity: 0; }
  to   { opacity: 1; }
}
.box {
  animation: fadeIn 2s ease-in-out;
}

Advanced Styling Effects

  • box-shadow, text-shadow :contentReference[oaicite:15]{index=15}
  • filter: blur(5px);, mix-blend-mode: multiply; :contentReference[oaicite:16]{index=16}
  • clip-path: circle(50%);

Typography Features


h1 {
  font-feature-settings: 'smcp' 1;
  text-shadow: 2px 2px 4px rgba(0,0,0,0.3);
}

Control OpenType features with font-feature-settings :contentReference[oaicite:17]{index=17}.

Form Styling

Customize checkboxes, radio buttons, selects, etc. :contentReference[oaicite:18]{index=18}.


input[type="checkbox"] {
  accent-color: #007bff;
}

Smooth Scrolling

Enable native smooth jumps with scroll-behavior :contentReference[oaicite:19]{index=19}.


html {
  scroll-behavior: smooth;
}
